# USA Census Intelligence Dashboard

A professional web GIS application built with 
ArcGIS Maps SDK for JavaScript 4.x

## 🌐 Live Demo
[View on GitHub Pages](https://nandinipulivarthi57.github.io/usa-census-intelligence-dashboard)

## 📸 Screenshots
![Dashboard Screenshot](screenshots/dashboard.png)

## ✨ Features
- 4 live data layers — States, Counties, Highways, Cities
- Dynamic renderer — Population Density, Total Population, 
  Median Age, Households
- Scale-dependent Arcade labeling
- City clustering with Heat Map toggle
- Sidebar search with attribute and spatial query
- Select by geometry — Rectangle, Circle, Polygon, Point
- Proximity analysis using GeometryEngine buffer
- Measure tool — Distance and Area
- Identify tool
- Basemap Gallery
- LayerList with collapsible legend

## 🛠️ Technologies
- ArcGIS Maps SDK for JavaScript 4.29
- ArcGIS REST Feature Services
- Arcade Expression Language
- HTML5, CSS3, JavaScript

## 📊 Data Sources
- Esri Sample Server 6 — USA MapServer
- Esri Sample Server 6 — Census MapServer
